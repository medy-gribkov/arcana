# Claude Skills

Cross-platform [Agent Skills](https://docs.anthropic.com/en/docs/agents/skills) for Claude Code, Antigravity, Codex CLI, and other compatible AI coding tools.

## Skills

| Skill | Description |
|-------|-------------|
| [project-migration](skills/project-migration/) | Migrate project folders while preserving Claude Code session data |

## Installation

```bash
# Install via Claude Code
/install mahdy-gribkov/claude-skills
```

## Compatibility

These skills follow the open Agent Skills standard (December 2025) and work with:
- Claude Code (Anthropic)
- Antigravity (Google)
- Codex CLI (OpenAI)
- Gemini CLI (Google)
- Cursor AI
