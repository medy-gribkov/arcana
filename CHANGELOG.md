# Changelog

## 2.0.0 (2026-02-13)

Major release. Added 12 new skills, renamed repo to Arcana, added npm CLI.

### Added
- codebase-dissection: systematic codebase analysis methodology
- code-reviewer: review local changes or remote PRs
- frontend-code-review: frontend file review with checklists
- frontend-design: distinctive frontend interfaces
- golang-pro: modern Go 1.21+ patterns and concurrency
- typescript: strict TypeScript style guide
- docx: Word document creation and manipulation
- xlsx: spreadsheet creation and manipulation
- remotion-best-practices: Remotion video creation in React
- skill-creator: create and update agent skills
- skill-creation-guide: how to build skills from scratch
- find-skills: discover and install skills from the ecosystem

### Changed
- Renamed repo from claude-skills to arcana
- Updated marketplace.json to version 2.0.0
- Added CONTRIBUTING.md, CODE_OF_CONDUCT.md, SECURITY.md
- Added GitHub issue and PR templates
- Added FUNDING.yml for GitHub Sponsors, Buy Me a Coffee, Ko-fi
- Enhanced CI validation (description quality checks)
